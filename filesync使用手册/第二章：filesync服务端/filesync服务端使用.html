2.2.1 环境准备

1.操作系统要求

  • g++ version >= 4.8

2.2.2 安装过程

  • 使用root登录linux系统;
  • 将压缩包filesync_x.x.x.tar.gz保存到任意一临时目录;
  • 解开压缩包'tar zxvf filesync_x.x.x.tar.gz';
  • 'cd filesync_x.x.x'目录,并执行'./install.sh'根据提示完成安装。

2.2.3 启动和停止

假设filesync安装在/usr/local/filesync目录下:

1.filesync以服务端模式启动filesync

前端运行命令,在终端输入: 'cd /usr/local/filesync && ./filesync server filesync.conf'

后台运行命令,在终端输入: 'cd /usr/local/filesync && nohup ./filesync server filesync.conf > filesync_server.logs'

2.停止filesync服务端模式

在终端输入:'ps -ef | grep "filesync server" | grep -v grep | awk '{print $2}' | xargs kill -9'

2.2.4 确定filesync以服务端模式已经启动

在终端输入:'netstat -an | grep -i listen | grep 10086',应该能看到10086端口正处于监听状态。
